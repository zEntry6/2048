What is this?

It's an online game that was created in March 2014 by the talented 19-year-old Italian web developer, Gabriele Cirulli. The aim of the game is to combine certain numbered squares to form the final result, 2048. It's a bit like sliding box guessing, and it's very similar to the app Threes!

How do I play?

Tiles with matching number values can be merged into a single tile, which receives the values' sum.

To move the board, use the directional arrows - or swipe.
